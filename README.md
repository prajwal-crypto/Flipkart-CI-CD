# 🛍️ Flipkart Clone – Dockerized CI/CD Project

A full-stack Flipkart-inspired e-commerce web application built using Flask and deployed using Docker and Render.

## 🚀 Live Demo

🔗 Live Website: https://flipkart-ci-cd.onrender.com

## 📌 Project Overview

This project demonstrates modern DevOps practices by combining application development, containerization, version control, and cloud deployment.

Users can browse products, search items, add products to a cart, and experience a responsive e-commerce interface.

## 🐳 Run Locally

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Flipkart-CI-CD.git
cd Flipkart-CI-CD
```

### Run Using Docker

```bash
docker-compose up --build
```

Open:

```text
http://localhost:5000
```

---

## 🧪 Run Tests

```bash
cd app
python -m pytest test_app.py -v
```

---

## ☁️ Deployment

The application is deployed on Render using Docker.

Deployment Workflow:

GitHub → Render → Docker Build → Live Application

Every new push to the main branch can be automatically deployed.
